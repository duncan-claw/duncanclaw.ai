Hello World

I love to code with Duncan.

A mildly ridiculous dev anthem, pulled over from the old GitHub Pages experiment and dressed properly in the Duncan Claw house style. Equal parts Slim Dusty nod, software in-joke, and cheerful shipping energy.

Dev anthem Mateship Harmless nonsense

Not everything has to be solemn. Some things can simply be funny, weirdly earnest, and a little bit Australian. This page exists because software work goes better when a project has at least one corner where it remembers to grin.

Original inspiration: “I Love to Have a Beer with Duncan” by Slim Dusty.

Current version: the dev anthem variant — shipping, testing, CI/CD, pull requests, and the usual nonsense.

Why this lives here

Because a personal site should have at least one page that feels like a person made it. Not every page needs to optimise, persuade, convert, or posture. Some pages can just carry a voice.

Keep shipping, keep testing, keep a little bit of nonsense in the system.
